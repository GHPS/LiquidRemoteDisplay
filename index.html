Liquidremotedisplay

Use an Arduino board to display PC information on an external LC display and control the machine with a remote control

Download Download View Liquidremotedisplay on GitHub

LiquidRemoteDisplay is a project that takes an Arduino plus LCD to use it as an external display and as a programmable remote control for an ordinary PC. Technically speaking it's a software emulator for a Matrix Orbital display with keypad - but the potentialities are enormous...

The project is useful and fun - plus easy to realize and expandable.

More info can be found in the official wiki: http://github.com/GHPS/LiquidRemoteDisplay/wiki
